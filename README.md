# Gerenciador de gastos usando React
[![NPM](https://img.shields.io/npm/l/react)](https://github.com/rodrigolb01/Expenses-tracker/blob/main/LICENSE) 

# Sobre o projeto

Gerenciador de gastos em que o usuário pode criar ações de débito ou crédito, no menu principal aparece uma fatura do total de despezas.

# Tecnologias utilizadas
- React js

# Como executar o projeto

Pré-requisitos: 
- npm 

```bash
# clonar repositório
git clone https://github.com/rodrigolb01/Expenses-tracker

# iniciar projeto
npm start
```
